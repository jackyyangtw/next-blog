# 常用 Type

## defineField 常用 type 說明

- `string`：單行短字串（如標題、名稱）
- `text`：多行長文字（如描述、備註）
- `number`：數字（如排序、價格）
- `boolean`：布林值（如啟用/停用）
- `date`：日期（如發佈日）
- `datetime`：日期時間（如活動時間）
- `image`：圖片（可上傳圖片）
- `file`：檔案（可上傳任意檔案）
- `url`：網址（如外部連結）
- `slug`：網址片段（通常自動產生，SEO 友善）
- `reference`：參照其他文件（如作者、分類）
- `array`：陣列（可搭配其他 type 使用，如圖片集、標籤）
- `block`：富文本（需自訂 schema，支援多樣內容）

| `slug`       | 網址片段，通常自動產生                    | SEO 友善網址            |
| `string`     | 單行短字串                                | 標題、名稱              |

| `number`     | 數字                                      | 價格、排序              |
| `boolean`    | 布林值                                    | 啟用/停用               |
| `date`       | 日期                                      | 發佈日                  |

| `url`        | 網址                                      | 外部連結                |
